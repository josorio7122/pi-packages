# ATS Keyword Optimization Guide

Detailed reference for keyword strategy. Loaded on-demand.

## Keyword Tiers

| Tier | Weight in ATS | Examples | Strategy |
|------|---------------|----------|----------|
| Hard Skills | 60-70% | Python, Salesforce, Six Sigma, GAAP, PMP | Extract 10-15 from JD; place in summary + bullets + skills |
| Job Scope | 20-30% | "P&L Management", "Team of 15", "M&A Integration" | Mirror exact language from posting |
| Soft Skills | 5-10% | Leadership, collaboration, communication | Include ONLY if explicitly in JD; pair with examples |

## Keyword Extraction Process

1. Copy the entire job description
2. Highlight ALL noun phrases (tools, methodologies, certifications)
3. Identify the top 20 most important terms
4. Create a mapping table:

| Exact Term from JD | Synonym/Variant | Your Experience |
|--------------------|-----------------|-----------------|
| Prompt Engineering | AI prompt design, LLM optimization | Developed GPT-4 prompts for customer service |
| Cloud-native | Cloud architecture, AWS/GCP | Built microservices on AWS ECS |

## Keyword Placement Strategy

| Section | Keywords to Include | How |
|---------|--------------------|----|
| Professional Summary | 5-7 primary keywords | Woven naturally into sentences |
| Core Skills | 15-25 keywords | Flat list, pipe-separated |
| Each work bullet | 1-2 keywords | Embedded in CAR-format achievements |
| Education | Degree name keywords | Full degree name spelled out |
| Certifications | Cert name keywords | Full official name + acronym |

## Common Keyword Mistakes

- ❌ **Keyword stuffing** — repetition > 2% triggers AI penalties
- ❌ **Only abbreviations** — use "SEO" AND "Search Engine Optimization"
- ❌ **Generic synonyms** — use exact terms from the posting
- ❌ **Listing skills you don't have** — dishonest and fails interviews
- ❌ **Ignoring soft skills entirely** — include if JD mentions them explicitly

- Cloud: cloud-native, Kubernetes, serverless, infrastructure as code

## Match Score Calculation

```
Match Score = (Matched Keywords / Total JD Keywords) × 100

Target:
- Minimum: 75%
- Competitive: 85%+
- Ideal: 90%+
```

## After Optimization

Recommend users validate with external tools:
1. Upload optimized resume to Jobscan, Teal, or ResumeWorded
2. Paste the target job description
3. Verify score meets 75%+ threshold
4. Iterate on missing keywords until target reached
