# Interface Design

Design engineering skill for pi. Build interfaces with craft, memory, and consistency.

## What It Does

Guides the creation of professional, consistent interface designs — dashboards, admin panels, SaaS apps, tools, and interactive products. NOT for marketing sites or landing pages.

- **Craft** — Principle-based design that produces professional, polished interfaces
- **Memory** — Save decisions to `.interface-design/system.md`, load automatically across sessions
- **Consistency** — Every component follows the same principles throughout

## Installation

This skill is part of [@josorio/pi-packages](https://github.com/josorio/pi-packages). It's auto-loaded when pi detects interface design work.

## Usage

The skill activates when building UI components, dashboards, or application interfaces. It guides design decisions around:

- Color palette and depth strategy
- Spacing systems and typography
- Surface elevation and border treatment
- Component patterns (buttons, cards, inputs, tables)
