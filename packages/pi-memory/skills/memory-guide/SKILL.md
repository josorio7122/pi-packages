# Memory Guide

You have access to long-term memory tools. Use them proactively.

## When to use memory_recall
- At the start of any task, search for relevant user preferences
- When the user references something they "mentioned before"
- When context from a previous session might be relevant

## When to use memory_store
- User explicitly asks you to remember something
- User states a clear preference ("I always prefer X")
- A key decision is made ("we decided to use Y")
- Contact information is shared

## When to use memory_forget
- User asks you to forget something
- Stored information is confirmed incorrect

## Memory categories
`preference`, `fact`, `decision`, `entity`, `other` — choose the most specific.
