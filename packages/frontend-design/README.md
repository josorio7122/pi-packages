# Frontend Design

Frontend design skill for pi. Create distinctive, production-grade frontend interfaces with high design quality.

## What It Does

Guides the creation of visually striking, memorable frontend interfaces that avoid generic AI aesthetics. Covers landing pages, marketing sites, web apps, and any UI where strong visual identity matters.

- **Bold aesthetic direction** — commit to a clear conceptual vision before coding
- **Distinctive typography** — unexpected, characterful font choices over generic defaults
- **High-impact motion** — orchestrated animations, scroll-triggered effects, surprising hover states
- **Creative composition** — asymmetry, overlap, grid-breaking elements, atmospheric backgrounds

## Installation

This skill is part of [@josorio/pi-packages](https://github.com/josorio/pi-packages). It's auto-loaded when pi detects frontend design work.

## Usage

The skill activates when building frontend interfaces, pages, or components where visual identity and design quality matter. It pushes for intentional, distinctive choices over safe defaults.
